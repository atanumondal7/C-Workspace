# C-Workspace

## Coded in Embarcadero

This repository contains the projects and ideas that I have built upon using C. These are fully functioning and ready to be compiled in any C compiler.

### List of Programs:

1. 4 Bit Adder - Functions exactly like a logical 4 Bit Adder with a proper carry logic. Mimics the actual flow of data in a digital adder.
2. Guess The Number - A guessing game where the user needs to guess a randomly generated number within a specific range of their choice. It also contains a fully automated scoreboard for displaying the number of attempts at end of each round along with an embedded ranking system.
